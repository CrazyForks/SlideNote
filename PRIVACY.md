# SlideNote 隐私政策

**生效日期**: 2025年1月12日
**版本**: 1.0

---

## 概述

SlideNote（"我们"、"本扩展"）致力于保护您的隐私。本隐私政策说明了我们如何收集、使用和保护您的信息。

---

## 数据收集

### 我们收集什么数据？

SlideNote **不收集**任何个人身份信息。您的所有数据均存储在本地设备上。

### 存储的数据内容

- **笔记内容**：您创建的笔记标题和正文
- **笔记元数据**：创建时间、修改时间、笔记排序
- **用户偏好**：当前选中的笔记ID

---

## 数据使用

### 我们如何使用您的数据？

您的数据仅用于：
- 在扩展内显示和管理您的笔记
- 通过 Chrome Storage Sync API 在您的设备间同步笔记

### 我们**不会**：

- 出售、租赁或共享您的数据给第三方
- 使用您的数据进行广告投放
- 分析您的个人内容
- 向任何外部服务器传输您的数据（除 Chrome 同步服务外）

---

## 数据存储

### 本地存储

所有笔记数据默认存储在您的浏览器本地存储中。

### 云端同步

如果您启用了浏览器同步功能，笔记数据将通过 **Chrome Storage Sync API** 在您的设备间同步。该服务由 Google 提供，数据在传输和存储过程中会被加密。

**重要说明**：Chrome Storage Sync 有存储容量限制（约 100KB）。如果数据量过大，同步可能失败。

---

## 数据删除

### 删除笔记

您可以通过扩展内的删除功能随时删除笔记。删除后数据将从本地存储中移除。

### 卸载扩展

卸载 SlideNote 扩展时，所有存储在本地和通过 Chrome 同步服务存储的数据将被删除。

---

## 第三方服务

### Chrome Storage Sync API

我们使用 Google 的 Chrome Storage Sync API 实现跨设备数据同步。Google 的隐私政策适用于该服务：
https://policies.google.com/privacy

### 无其他第三方服务

SlideNote **不使用**任何其他第三方服务，包括但不限于：
- 分析工具
- 广告服务
- 社交媒体插件
- 用户追踪工具

---

## 权限说明

### `storage` 权限

用于存储和同步您的笔记数据。

### `sidePanel` 权限

用于在浏览器侧边栏显示笔记面板。

---

## 儿童隐私

SlideNote 不针对 13 岁以下儿童。我们不会故意收集儿童的个人信息。
