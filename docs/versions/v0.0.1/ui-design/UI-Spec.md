## 一、设计原则

| 原则 | 说明 |
|------|------|
| **克制的配色** | 单色主调，不抢注意力 |
| **清晰的层级** | 列表 vs 内容，一目了然 |
| **舒适的间距** | 呼吸感，不拥挤 |
| **统一的圆角** | 温和不尖锐 |

## 三、视觉规范

### 3.1 色彩系统

```css
/* 基础色彩 */
--color-bg-primary:     #FFFFFF;    /* 主背景 */
--color-bg-secondary:   #F5F5F5;    /* 次级背景 */
--color-bg-hover:       #E8E8E8;    /* 悬停背景 */
--color-bg-active:      #E0E0E0;    /* 激活背景 */

/* 文字色彩 */
--color-text-primary:   #1A1A1A;    /* 主文字 */
--color-text-secondary: #666666;    /* 次级文字 */
--color-text-tertiary:  #999999;    /* 辅助文字 */
--color-text-placeholder: #BBBBBB;  /* 占位文字 */

/* 边框与分割线 */
--color-border-light:   #E5E5E5;    /* 浅边框 */
--color-border-medium:  #D0D0D0;    /* 中边框 */

/* 功能色彩 */
--color-primary:        #0066CC;    /* 主色（仅交互状态） */
--color-success:        #22C55E;    /* 成功 */
--color-error:          #FF4444;    /* 错误/警告 */
--color-info:           #0066CC;    /* 信息 */
```

### 3.2 字体系统

```css
/* 字体家族 */
--font-family-base: -apple-system, BlinkMacSystemFont, "Segoe UI",
                    "Helvetica Neue", Arial, sans-serif;

/* 字号 */
--font-size-xs:   11px;   /* 辅助信息、预览文字 */
--font-size-sm:   12px;   /* 次级文字、列表标题 */
--font-size-base: 13px;   /* 基础文字 */
--font-size-md:   14px;   /* 标题 */
--font-size-lg:   16px;   /* 大标题 */
--font-size-xl:   20px;   /* 特大标题 */

/* 行高 */
line-height: 1.5;         /* 全局行高 */
textarea: line-height: 1.7; /* 编辑器行高 */

/* 字重 */
--font-weight-normal:  400;
--font-weight-medium:  500;
--font-weight-semibold: 600;
```

### 3.3 间距系统

```css
--spacing-xs:  3px;    /* 极小间距 */
--spacing-sm:  6px;    /* 小间距 */
--spacing-md:  12px;   /* 中等间距 */
--spacing-lg:  18px;   /* 大间距 */
--spacing-xl:  24px;   /* 超大间距 */
```

### 3.4 圆角系统

```css
--radius-sm: 4px;      /* 小圆角 */
--radius-md: 6px;      /* 默认圆角 */
--radius-lg: 8px;      /* 大圆角 */
```

### 3.5 阴影系统

```css

--shadow-md:  0 4px 12px rgba(0, 0, 0, 0.08);
--shadow-lg:  0 8px 24px rgba(0, 0, 0, 0.12);
```

## 四、组件设计

```
┌─────────────────────────────────────┐
│ 🔍 搜索笔记...                      │
└─────────────────────────────────────┘

高度: 36px
圆角: 8px
背景: #F5F5F5
边框: 1px solid #E5E5E5
图标: 16px, #999999
```

**状态：**
- 默认：灰底，浅边框
- 聚焦：白底，蓝色边框
- 有内容：显示清除按钮

### 4.2 笔记列表项

```
未选中：
  ┌───────────────────────────────┐
  │ 常用账号               [...]  │
  │ Gmail, AWS, 阿里云...          │
  └───────────────────────────────┘

选中：
  ┌───────────────────────────────┐
  │ 常用账号               [...]  │  ← 蓝色左侧条
  │ Gmail, AWS, 阿里云...          │
  └───────────────────────────────┘
      ↑ 背景 #F0F7FF
```

**尺寸：**
- 高度：60px（单行） / 自适应（多行）
- 内边距：12px 16px
- 间距：0（列表项无间距，分割线分隔）

**状态：**
- 默认：白底
- 悬停：#F5F5F5
- 选中：#F0F7FF + 3px 蓝色左侧条

### 4.3 新建按钮

```
┌──┐
│  +  │  图标按钮 (30x30px)
└──┘
```

**尺寸：** 30x30px，正方形图标按钮

**位置：** 搜索栏右侧，与搜索框同高

**状态：**
- 默认：白底，蓝字，浅边框
- 悬停：浅蓝底 #F0F7FF，蓝色边框
- 点击：深蓝底 #0066CC，白字

**图标：** 加号 (+)，14x14px SVG

## 九、动效规范

### 9.1 动画时长

```css
--duration-fast:   150ms;   /* 状态切换 */
--duration-base:   200ms;   /* 基础动画 */
--duration-slow:   300ms;   /* 进场动画 */
```

### 9.2 缓动函数

```css
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
--ease-out:   cubic-bezier(0.0, 0, 0.2, 1);
--ease-in:    cubic-bezier(0.4, 0, 1, 1);
```

### 9.3 关键动效

| 交互 | 动效 | 时长 |
|------|------|------|
| 侧边栏展开 | 从右侧滑入 + 渐显 | 200ms |
| 笔记切换 | 内容淡入 | 150ms |
| 删除确认 | 弹窗缩放淡入 | 200ms |
| 已保存提示 | 底部滑入 + 淡出 | 2000ms |

## 十、可访问性

### 10.1 键盘导航

- `Tab`: 焦点在控件间移动
- `Enter`: 确认/新建
- `Escape`: 取消/关闭
- `Cmd/Ctrl + F`: 聚焦搜索框

### 10.2 焦点状态

所有可交互元素必须有清晰的焦点状态：

```css
:focus-visible {
  outline: 2px solid #0066CC;
  outline-offset: 2px;
}
```
