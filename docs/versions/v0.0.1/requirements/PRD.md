## 一、产品概述

### 1.1 产品定义

SlideNote（侧边笔记）是一款 Chrome 侧边栏笔记插件，专为"需要跨设备随手存、随时查"的碎片信息而生。

### 1.2 产品定位

**"Chrome 侧边栏里的便利贴"** — 不是笔记应用，而是随身碎片信息夹。

### 1.3 产品原则

| 原则 | 解释 |
|------|------|
| **极简至上** | 只做一件事：存储和同步短文本 |
| **秒开即用** | 点击图标，内容立即可见 |
| **无声打扰** | 不弹通知、不强引导 |
| **稳定可靠** | 数据不丢失，同步不失败 |
| **永不过时** | 设计克制，十年后依然好用 |

## 二、用户与场景

### 2.1 目标用户

- **多设备工作者**：公司 Mac + 家里 Mac + 笔记本
- **技术人员**：开发、运维、产品经理
- **运营/自媒体**：多账号管理

### 2.2 核心场景

| 场景 | 描述 |
|------|------|
| 场景1 | 公司电脑上存的阿里云密码，回家电脑上要登录 |
| 场景2 | 把团队的 API Key 存这里，换电脑还能用 |
| 场景3 | 临时记一个服务器地址，下次打开还在 |
| 场景4 | 把常用的几个账号集中放一起，复制粘贴 |

## 三、功能需求

### 3.1 功能边界

**✅ 本版本包含：**

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 创建笔记 | 点击 + 号，输入标题，开始写作 | P0 |
| 编辑笔记 | 点击内容区，直接编辑，自动保存 | P0 |
| 删除笔记 | 删除按钮，二次确认 | P0 |
| 笔记列表 | 左侧显示所有笔记，点击切换 | P0 |
| 搜索笔记 | 顶部搜索框，实时过滤 | P0 |

**❌ 本版本不包含：**

- 笔记分类/标签
- 富文本/图片
- 导入/导出
- 加密
- 历史版本
- 协作分享

### 3.2 功能详述

#### 3.2.1 创建笔记

- 入口：搜索栏右侧的 "+" 图标按钮
- 行为：
  - 创建新笔记，默认标题 "未命名笔记"
  - 光标自动定位到标题处
  - 笔记自动添加到列表顶部

#### 3.2.2 编辑笔记

- 入口：点击笔记列表项或内容区
- 行为：
  - 标题和内容均可直接编辑
  - 停止输入 1 秒后自动保存
  - 显示 "已保存" 提示（轻量级，不弹窗）

#### 3.2.3 删除笔记

- 入口：笔记列表项的删除图标
- 行为：
  - 弹出确认框："确定删除「xxx」？"
  - 3 秒后确认按钮可点击（防误操作）
  - 删除后自动选中下一条笔记

#### 3.2.4 搜索笔记

- 入口：顶部搜索框
- 行为：
  - 实时过滤笔记列表（匹配标题和内容）
  - 匹配文字高亮显示
  - 清空搜索框恢复完整列表


## 五、非功能需求

### 5.1 性能要求

| 指标 | 要求 |
|------|------|
| 打开速度 | < 100ms |
| 搜索响应 | < 50ms |
| 保存延迟 | 1 秒（自动保存） |

### 5.2 兼容性要求

| 平台 | 要求 |
|------|------|
| Chrome | ≥ 88 版本 |
| Edge | ≥ 88 版本（Chromium 内核） |
| 系统平台 | macOS / Windows / Linux |

### 5.3 数据要求

- 使用 Chrome Storage API (storage.sync)
- 单条数据限制：约 8KB
- 总容量限制：约 100KB
- 自动跨设备同步
