## 版本概述

v0.0.2 版本在 v0.0.1 MVP 基础上，优化用户体验，新增侧边栏折叠和输入交互优化。

### 主要目标

- 提升编辑体验，减少操作步骤
- 增加侧边栏折叠功能，最大化编辑空间
- 完善开发流程，区分开发/生产构建

## 已实现功能详情

### 侧边栏折叠/展开

- **设计**: 分隔线上的微型控制器
- **功能**:
  - 点击右边缘中央的收起按钮折叠侧边栏
  - 折叠后侧边栏宽度从 180px 变为 40px
  - 折叠状态点击中央圆形按钮展开
  - 新建笔记或搜索时自动展开
  - 状态持久化到 Chrome Storage

### 新建笔记自动聚焦标题

- **功能**: 创建新笔记时自动聚焦标题输入框
- **优化**: 用户可直接输入标题，无需手动点击
- **交互**: Tab 键在标题和内容之间快速切换

### 开发/生产构建分离

- **开发版**: 橙色图标，名称显示 "Dev"，可同时安装
- **生产版**: 深色图标，标准名称
