## 投稿标题

**SlideNote - 住在浏览器侧边的碎片信息收集箱**

---

## 项目简介（100字以内）

SlideNote 是一款 Chrome 浏览器侧边栏笔记插件。它利用 Chrome 114+ 的 Side Panel API，将笔记面板固定在浏览器侧边，让你在浏览网页的同时随时记录和查看碎片信息（API Key、服务器地址、快捷命令等）。支持跨设备自动同步，无框架依赖，打包仅 20KB。

---

## 为什么做这个项目？

写代码时经常需要临时查一些命令或配置，查完切回编辑器就忘了；在公司电脑学了个新命令，回家想用却找不到；各种云服务的密钥、服务器地址散落各处，每次都要翻半天。

现有的笔记插件都悬浮在页面上，遮挡内容，打断工作流。直到发现 Chrome 的 Side Panel API，可以固定在侧边栏不干扰主页面，于是做了 SlideNote。

---

## 技术亮点

| 特点 | 说明 |
|------|------|
| **Vanilla JS** | 无框架依赖，ES6+ 原生开发 |
| **20KB 打包** | 极致轻量，加载 < 100ms |
| **Chrome Storage Sync** | 免费云同步，无需服务器 |
| **冲突处理** | 时间戳策略，多设备自动合并 |
| **UI 设计** | CSS Variables 设计系统，自适应宽度 |

---

## 为什么不用框架？

这类 CRUD 功能不需要 React/Vue。用框架：
- 打包后至少多 100KB
- JSX 编译、虚拟 DOM diff 都是额外开销
- 框架升级风险，代码三年后还能跑吗？

原生 JS 足够简单，状态管理用个 EventBus，渲染用个 `innerHTML` 或 `createElement`，搞定。
