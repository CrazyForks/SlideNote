## 一句话介绍

Chrome 浏览器侧边栏笔记插件，跨设备自动同步，专注碎片信息存储。

---

## 产品定位

```
"Chrome 侧边栏里的便利贴"
```

不是又一个笔记应用，而是：
- 随手存放碎片信息（密码、API Key、服务器地址）
- 打开浏览器就能看到，点击即用
- 多台设备之间自动同步

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 侧边栏展示 | Chrome 114+ Side Panel API，固定在浏览器侧边 |
| 自动保存 | 停止输入 1 秒后自动保存 |
| 跨设备同步 | 基于 Chrome Storage Sync API，自动云端同步 |
| 搜索过滤 | 实时搜索标题和内容 |
| 删除确认 | 3 秒倒计时防误操作 |

## 技术亮点

### 极简技术栈

```
Vanilla JS (ES6+)  →  无框架依赖
Vite               →  快速构建
CSS Variables      →  设计系统
Chrome Storage     →  免费云同步
```

### 为什么不用框架？

| 理由 | 说明 |
|------|------|
| 性能 | 加载时间 < 100ms，无 ~100KB+ 框架开销 |
| 简单 | CRUD 功能不需要复杂状态管理 |
| 稳定 | 无框架升级风险，代码十年后依然可用 |
| 可控 | 完全掌控渲染逻辑，调试更直接 |

### 核心架构

```
┌─────────────────────────────────────┐
│              App                    │
│  ┌──────────┐  ┌──────────────────┐ │
│  │  Store   │←→│  SyncManager     │ │
│  │          │  │  (冲突处理)       │ │
│  └────┬─────┘  └──────────────────┘ │
│       │                             │
│       └─────→ EventBus              │
│               ↓                     │
│       ┌───────┴───────┐             │
│       │  Components    │             │
│  NoteList │ NoteEditor │ SearchBar   │
└─────────────────────────────────────┘
```

**冲突解决策略**：
1. 本地有未提交变更 → 保留本地
2. 远程时间戳更新 → 使用远程
3. 远程新增 → 自动合并

## 设计理念

### 视觉风格

```
克制的配色    → 单色主调，不抢注意力
清晰的层级    → 列表 vs 内容，一目了然
舒适的间距    → 呼吸感，不拥挤
统一的圆角    → 温和不尖锐
```

### 布局规范

```
侧边栏宽度: 480px
笔记列表:   180px（紧凑）
笔记内容:   300px（flex: 1）
```

### 尺寸系统

```css
字号: 11/12/13/14/16px
间距: 3/6/12/18/24px
圆角: 4/6/8px
```

## 数据存储设计

### Chrome Storage Sync API

```javascript
// 活跃笔记（同步，100KB 限制）
{
  slidenote_notes: [
    { id, title, content, createdAt, updatedAt }
  ],
  slidenote_active_id: "note_xxx"
}

// 归档笔记（v0.0.2，本地，5MB+）
{
  slidenote_archived_notes: [...]
}
```

### 存储限制应对

| 策略 | 版本 |
|------|------|
| 容量监控警告 | v0.0.1 |

## 安装使用

### 开发者安装

```bash
git clone https://github.com/maoruibin/SlideNote.git
cd slide-note
npm install
npm run build
```

然后在 Chrome 中加载 `dist` 目录作为未打包的扩展。

### 用户安装（计划）

Chrome Web Store 搜索 "SlideNote" 一键安装
